Accessibility Information

This app is built to support:

  • Keyboard-only navigation for all actions/forms/tables
  • High-contrast colors and dark mode for low-vision users
  • ARIA labels for important controls, headings, actions
  • Screen reader compatibility for all pages
  • Responsive layouts for mobile/tablet/computer use
For accessibility audits, use online tools like WAVE or axe DevTools.
