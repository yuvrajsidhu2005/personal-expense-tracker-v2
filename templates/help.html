Help & FAQ

  • How do I use this expense tracker?
    You can add, edit, and delete expenses. Analytics are shown on the dashboard. Expenses are temporary if you're a guest, or saved permanently if you create an account.
  • How do I use tags?
    Type tags (#food, #travel) in the tags field when adding expenses or income. Use the tags search in All Expenses to filter quickly.
  • How do I set themes and budgets?
    You can pick a theme and set your monthly budget in Profile Settings after logging in.
  • How do I print or export my expenses?
    Go to All Expenses, use the Print or Export CSV buttons. As a guest, you can still print/export all current session data.
  • Can I restore my data if I log out?
    For permanent save/restore, sign up or log in.
  • Accessibility Features?
    All pages support keyboard navigation, labels, and dark mode. More info in the Accessibility section.
